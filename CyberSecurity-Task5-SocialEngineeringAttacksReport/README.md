# Task 5: Research Report — Social Engineering Attacks
 
##  Project Overview
 
This folder contains my comprehensive research report on Social Engineering attacks, completed for the Oasis Infobyte Cyber Security internship. The report analyzes vulnerabilities within the "human firewall" and how psychological manipulation is used by adversaries to circumvent technical security controls.
 
##  Contents
 
- `social_engineering_report.md`: The full research report covering Phishing, Pretexting, Baiting, and Quid Pro Quo attacks. It features real-world case studies, a threat comparison matrix, and a recommended 5-point employee security awareness training checklist.
##  Key Learnings
 
- Recognizing the psychological levers—such as fear, urgency, curiosity, and trust—that are commonly exploited by attackers.
- Analyzing high-profile breaches (e.g., the 2020 Twitter Hack and Stuxnet) that relied heavily on social engineering vectors to gain initial access.
- Developing effective, human-centric countermeasures, including non-phishable MFA (FIDO2), strict out-of-band verification protocols, and continuous, role-based employee training.
