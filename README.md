##  About This Repository
This repository contains my practical and research-based task submissions for the Oasis Infobyte System/Security Internship Program (OIBSIP). The tasks demonstrate a balanced approach to the Cyber Security domain, covering both hands-on technical execution (scanning and exploitation) and professional security analysis (threat reporting and mitigation strategies).

##  Task Index

Below are the 4 tasks I selected and completed for this track. Click on any task to view the source files, raw data, and detailed documentation.

*  **[Task 1: Basic Network Scanning with Nmap](./CyberSecurity-Task1-NetworkScanning/)**
  * **Focus:** Network reconnaissance, port scanning, service versioning, and OS detection.
  * **Deliverables:** Raw scan logs, terminal screenshots, and port security analysis.

*  **[Task 3: SQL Injection on DVWA](./CyberSecurity-Task3-SQLInjection/)** 
  * **Focus:** Application-layer vulnerabilities, input manipulation, and parameterized query remediation.
  * **Deliverables:** Exploitation notes, payload logs, and mitigation strategies.

*  **[Task 4: Common Network Security Threats Report](./CyberSecurity-Task4-NetworkSecurityThreatsReport/)** 
  * **Focus:** Threat intelligence, DoS/DDoS, MITM attacks, IP spoofing, and enterprise mitigation.
  * **Deliverables:** Comprehensive markdown research report.

*  **[Task 5: Social Engineering Attacks Report](./CyberSecurity-Task5-SocialEngineeringAttacksReport/)** 
  * **Focus:** Human-centric vulnerabilities, phishing paradigms, pretexting, and security awareness training.
  * **Deliverables:** Comprehensive markdown research report and case studies.

##  Tools & Tech Stack Utilized
* **Scanning & Reconnaissance:** Nmap
* **Exploitation Environment:** DVWA (Damn Vulnerable Web Application) running on a local virtualized server
* **Virtualization:** Kali Linux VM, Metasploitable 2 (Target VM)
* **Documentation:** Markdown, Git/GitHub

*Note: All practical offensive security tasks within this repository were conducted strictly within isolated, locally hosted virtual machine environments in accordance with ethical hacking guidelines.*
