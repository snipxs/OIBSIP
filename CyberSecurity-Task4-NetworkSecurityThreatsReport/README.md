# Task 4: Research Report — Common Network Security Threats
 
##  Project Overview
 
This folder contains my research report on common network security threats, completed as part of the Oasis Infobyte Cyber Security internship. The report explores the technical mechanics, real-world impacts, and enterprise mitigation strategies for prevalent network-layer attacks.
 
##  Contents
 
- `network_security_threats_report.md`: The complete research report detailing DoS/DDoS, Man-in-the-Middle (MITM), IP Spoofing, and DNS Poisoning attacks, including a comparative threat matrix and key takeaways for network administrators.
##  Key Learnings
 
- Understanding the fundamental differences between resource exhaustion (DDoS) and packet manipulation (Spoofing/MITM) attacks.
- Analyzing real-world historical incidents (e.g., the 2016 Dyn Cyberattack and the 2011 DigiNotar Breach) to understand business impact.
- Formulating enterprise-grade defense strategies, including zero-trust architecture, robust encryption enforcement (HTTPS/IPsec), and network traffic filtering.
